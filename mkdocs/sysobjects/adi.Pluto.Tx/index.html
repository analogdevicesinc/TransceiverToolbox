    adi.Pluto.Tx

    adi.Pluto.Tx
    Receive data from Analog Devices AD9361 transceiver
    Description
    The adi.Pluto.Tx System object is a signal sink that can
    transmit complex data from the Pluto.

    tx = adi.Pluto.Tx;
    tx = adi.Pluto.Tx('uri','192.168.2.1');

    Product Page

    See also adi.AD9363.Tx
